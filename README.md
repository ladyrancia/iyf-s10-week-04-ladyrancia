# Week 04 - JavaScript Fundamentals

## 📌 Overview

This project covers the basics of JavaScript including variables, data types, functions, control flow, loops, arrays, objects, and a mini project.

---

## 🚀 What I Learned

* JavaScript variables (`let`, `const`)
* Data types and operators
* Functions (declarations, expressions, arrow functions)
* Control flow (`if/else`, `switch`)
* Loops (`for`, `while`)
* Arrays and array methods (`map`, `filter`, `reduce`)
* Objects and object methods
* Working with arrays of objects

---

## 🧮 Mini Project: Calculator

Built a console-based calculator that supports:

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%)
* Power (**)

Includes error handling for division by zero.

---

## 🎓 Final Project: Grade Tracker

Created a student grade management system that can:

* Add students
* Calculate averages
* Find top student
* Identify struggling students
* Generate report cards

---

## 🛠️ How to Run

1. Open `index.html` in Live Server
2. Open browser console (F12)
3. View outputs
